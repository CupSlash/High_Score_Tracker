# High Score Tracker 


## Description

Compete in games of Tic Tac Toe and number guessing with your friends!

## Getting Started

### Dependencies

uses python 3x. imports csv, random, and json. 

### Installing

no installing should be required, it is designed to be run in vs code

### Executing program

- click run button
- create account
- sign into account
- choose game
- play game
- see who got best score
