# High Score Tracker 


## Description

Compete in games of Tic Tac Toe with your friends or play a number guessing game!

## Getting Started

### Dependencies

any version of python 3. imports libraries csv, random, and json. 

### Installing

no installing should be required, it is designed to be run in vs code(or codespaces if you prefer).if you want to try and exxport it, be my guest, but it will be a long and harrowing journey

### Executing program

- Go to main page
- click run button
- create account
- sign into account
- play game
- see who got best score
