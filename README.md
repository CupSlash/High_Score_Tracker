# High Score Tracker 


## Description

Compete in games of Tic Tac Toe and guessing numbers with friends! create an account, sign in, and play against friends!

## Getting Started

### Dependencies

any version of python 3. imports libraries csv, random, and json. 

### Installing

no installing should be required, it is designed to be run in vs code.if you want to try and exxport it, be my guest, but it will be a long and harrowing journy

### Executing program

- click run button
- create account
- sign into account
- play game
- see who got best score
